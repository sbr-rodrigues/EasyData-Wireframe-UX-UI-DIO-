# 📊 EasyData – Wireframe UX/UI (DIO)

## 📌 Descrição do Projeto

Este projeto apresenta um **wireframe de média fidelidade de uma One Page** desenvolvido como parte do desafio de UX/UI da plataforma **Digital Innovation One**.

A proposta foi criar a estrutura de interface para um produto digital utilizando conceitos de **psicologia aplicada ao design e princípios de usabilidade**, garantindo uma navegação intuitiva e uma experiência clara para o usuário.

O produto escolhido foi **EasyData**, uma plataforma fictícia de ensino de **Data Analytics**, cujo objetivo é ensinar análise de dados do nível iniciante ao profissional.

---

# 🎯 Objetivos do Projeto

* Criar um **wireframe de média fidelidade**
* Aplicar **leis de usabilidade e psicologia do design**
* Estruturar uma **One Page intuitiva**
* Demonstrar organização de interface para **desktop e mobile**
* Documentar o projeto no **GitHub**

---

# 🧠 Conceitos de UX Aplicados

Durante o desenvolvimento foram aplicados princípios importantes de usabilidade:

### ✔ Visibilidade do sistema (Nielsen)

O usuário entende rapidamente o objetivo da página através de:

* título claro
* botão de ação **"Comece Agora"**
* hierarquia visual bem definida

### ✔ Consistência

Os elementos visuais mantêm padrões consistentes:

* tipografia
* espaçamento
* organização das seções

### ✔ Navegação intuitiva

A interface possui um **botão de voltar**, permitindo ao usuário retornar facilmente à navegação anterior, reforçando o princípio de **controle do usuário**, importante na usabilidade.

### ✔ Escaneabilidade da informação

O conteúdo foi dividido em seções claras:

* Hero (apresentação)
* Sobre o curso
* Benefícios
* Depoimentos
* Rodapé

Isso facilita a leitura rápida e melhora a experiência do usuário.

---

# 🖥 Estrutura da Página

A página foi estruturada como **One Page**, contendo:

1️⃣ **Header**

* Logo EasyData
* Menu de navegação (Menu | Sobre | Contato)

2️⃣ **Hero Section**

* Título principal
* Descrição do curso
* Botão de ação "Comece Agora"

3️⃣ **Sobre o Curso**

* Apresentação do curso de Data Analytics
* Explicação do conteúdo e ferramentas utilizadas

4️⃣ **Benefícios**

* Aprenda Python para dados
* Projetos práticos
* Certificado profissional

5️⃣ **Depoimentos**

* Feedback de alunos
* Avaliações positivas

6️⃣ **Navegação**

* Botão **Voltar** para facilitar o retorno do usuário

7️⃣ **Rodapé**

* Informações da plataforma
* Links para redes sociais
